# Godo

CLI interface for managing todolist.

## Quickstart

```
# Install

# Add in path
```

## Usage

```bash
# Add a todo
godo add "Do something"

# List todos
godo list [-a]

# Mark as done / undone
godo toggle [id id2 id3...]

# Edit
godo edit [id] [-m "New name"]
```
